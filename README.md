## Project
This project is built with Grails Framework ( Version 4.0.3 )
  ### Or
Download Grails from [netsansoftwarec.com](https://grails.org/index.html)

## Clone this project

```bash
$ git clone https://github.com/MahmoudAhmadOsman/grails-img-api.git
```

```bash
$ grails create-app 
```
